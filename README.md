Description: A Windows desktop application that captures ETW (Event Tracing for Windows) activity in real time and provides tools for diagnostics and troubleshooting.

Overview

- Real-time ETW event capture
- File, registry, process, and network monitoring
- Event filtering, searching, and history
- On-demand diagnostics (disk checks, large file scans, etc.)
- Optional analysis through an assistant
- Local storage of all settings and history
- Fully on-device operation with no external dependencies

Features
- Real-Time ETW Monitoring
- Tracks low-level Windows activity:
- File creation, deletion, and read/write operations
- Registry access and changes
- Process start/exit events
- Network events (where supported)
- Events appear immediately and can be filtered or searched.
- Diagnostics
- Built-in diagnostic tools:
- Disk space check
- Large file scan
- Additional checks depending on configuration
- Diagnostic results can be analyzed together with recent ETW activity.

AI Assistant
- Summarize recent system activity
- Interpret diagnostic results
- Identify unusual patterns
- Answer questions about captured events
- Depending on configuration, the assistant may use a local or external model.

Settings
- Monitoring Options
- Enable or disable capture for file, registry, process, and network events
- Skip events generated by the monitoring process
- Configure AI API Connectivity
